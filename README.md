# Countdown to July 18, 2026

A romantic countdown page with animations and music. Hosted on GitHub Pages.

## How to Set Up on GitHub Pages

1. **Create a GitHub Account**
   - Go to github.com and sign up (free)

2. **Create a New Repository**
   - Click the "+" icon in the top-right and select "New repository"
   - Name it: `countdown` (or any name you prefer)
   - Make it **Public**
   - Click "Create repository"

3. **Upload Your Files**
   - Click "Upload files" button
   - Drag and drop all files from this folder:
     - `index.html`
     - `me.gif`
     - `jaxi.gif`
     - `photo.png`
     - `music.mp3`
     - `README.md`
   - Commit the files

4. **Enable GitHub Pages**
   - Go to Settings → Pages
   - Under "Source", select "Deploy from a branch"
   - Select branch: `main`
   - Select folder: `/ (root)`
   - Click "Save"

5. **Get Your Link**
   - GitHub will show your site URL: `https://yourusername.github.io/countdown`
   - Share this link with anyone!
